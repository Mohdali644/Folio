## ⚡ Core Engine Architecture & Physics

The interaction layer of this portfolio is governed by a suite of bespoke, highly optimized Vanilla JavaScript algorithms, compartmentalized for maximum execution speed[cite: 6].

### 1. The Canvas Matrix (`Particle Physics Engine`)
The hero section features an interactive HTML5 `<canvas>` network built on custom 2D vector mathematics[cite: 4, 6]:
*   **Dynamic Density Generation:** Automatically calculates optimal particle density based on current viewport dimensions via the algorithm `(width * height) / 15000`[cite: 6].
*   **Euclidean Repulsion Mechanics:** Tracks real-time mouse coordinates (`e.clientX`, `e.clientY`) and applies an inverse-square repulsion force to particles entering a 150px radius[cite: 6].
*   **Proximity Node Linking:** Utilizes the Pythagorean theorem (`Math.sqrt(dx * dx + dy * dy)`) to calculate distances between drifting particles, drawing dynamic, opacity-faded connection lines using `ctx.lineTo()` when distance drops below 100px[cite: 6].

### 2. Spatial 3D Tilt Mechanics
Project cards and layout elements simulate physical depth using complex trigonometric calculations[cite: 6]:
*   Monitors `mousemove` events to calculate the exact cursor position relative to the element's center using `getBoundingClientRect()`[cite: 6].
*   Translates X/Y deltas into precise CSS `rotateX` (up to -15deg) and `rotateY` (up to 15deg) properties, mapped via a 1000px perspective lens for realistic hardware-accelerated 3D skewing[cite: 6].

### 3. Asynchronous Render Pipeline
Bypasses traditional, performance-heavy scroll listeners by utilizing native browser APIs[cite: 6]:
*   **IntersectionObserver API:** Monitors the DOM asynchronously to detect when elements breach a 15% visibility threshold (`threshold: 0.15`)[cite: 6].
*   **Web Animations API Sequencing:** Once triggered, elements are animated directly via the browser's compositor thread using the WAAPI (`element.animate()`), executing a seamless `translateY` and opacity fade without locking the main JavaScript thread[cite: 6].

### 4. Magnetic UI Interceptors
Buttons and holographic logos feature a custom magnetic pull[cite: 4, 6]:
*   Calculates the distance from the cursor to the center of the bounding box[cite: 6].
*   Applies a localized `translate(x, y)` transform multiplied by a 0.3 friction coefficient, physically dragging the element toward the user's touchpoint[cite: 6].
