## ⚡ Core Architecture & Engineering Specs

### 🧠 JavaScript Processing Matrix (`app.js`)
The interaction layer is governed by a suite of custom-built, highly optimized vanilla JavaScript algorithms[cite: 6]:
*   **HTML5 Canvas Particle Physics:** A dynamic, full-screen particle network that calculates real-time vector velocities[cite: 6]. It utilizes Euclidean distance formulas to create a proximity-based repulsion field relative to user mouse coordinates[cite: 6].
*   **Asynchronous Scroll Observers:** Implements the native `IntersectionObserver` API coupled with the Web Animations API to sequence high-performance, hardware-accelerated scroll reveals without locking the main thread[cite: 6].
*   **3D Trigonometric Tilt Engine:** Computes bounding rectangles (`getBoundingClientRect`) and mouse deltas to dynamically apply CSS `rotateX`, `rotateY`, and `scale3d` properties to project cards for a physical depth simulation[cite: 6].
*   **Magnetic Element Interceptor:** Custom tracking algorithms bind interactive UI nodes to cursor micro-movements, translating elements on a constrained X/Y axis for heightened tactile feedback[cite: 6].

### 🎨 Advanced Aesthetic Systems (`style.css`)
The visual engine is constructed using next-generation CSS properties, heavily utilizing custom CSS variables (`--bg-base`, `--accent-cyan`) for rapid theming[cite: 5]:
*   **Holographic Glassmorphism:** Deeply layered `backdrop-filter: blur(20px)` and semi-transparent RGBA properties create a frosted, cybernetic UI layer[cite: 5].
*   **Cinematic Sequencing:** Features a multi-stage loading wrapper and a 3D perspective moving background grid masked with linear gradients for spatial depth[cite: 5].
*   **Complex Keyframe Animations:** Utilizes over 15 custom `@keyframes` for effects ranging from cyberpunk glitch text, continuous gradient flows, and rotating laser borders, to SVG footer wave masking[cite: 5].
*   **Absolute Responsiveness:** Engineered with extreme precision using CSS Grid, Flexbox, and complex media queries to ensure flawless scaling from 4K ultra-wide monitors down to 380px mobile viewports[cite: 5].

## ⚙️ Deployment Protocol

### Local Initialization
To boot the engine locally, clone the repository and run it through a standard web server protocols to bypass localized CORS restrictions on module loading.

```bash

# 2. Navigate into the core directory
cd Portfolio-Engine

# 3. Launch via Live Server (VS Code) or Python simple HTTP server
